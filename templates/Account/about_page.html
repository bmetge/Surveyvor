Surveyvor is a lightweight web survey service developed by a team of NTU undergraduates. The website serves as a graded project for the Software Engineering module [Code CZ2006] under the team Surveyvor. The project was completed within a time frame of 10 weeks. This involved the documentation of system requirements, modeling classes and sequence diagrams, implementation of coding ,and testing. The core technologies involved are HTML programming, python and Django Web Framework. This web survey service platform also come up with an Android app, which can do same function as the website.
